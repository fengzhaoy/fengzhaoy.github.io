Axure使用笔记

2019-05-24

  1. 母版：可以快速设计出共用的按钮样式组件，当另一页面用到同样的组件时，直接拖动过去就可以实现效果。非常省时省力。
  2. 动态面板：可以实现弹出层的效果，也可以实现动态显示或者隐藏某一片区域。拖动一个动态面板到编辑区 然后双击选择 state 进行相应的内容设置。如果在设计中动态面板挡住了页面中的其他按钮时，则可以在 概要：页面窗口中 找到命名的动态面板 然后在后面有一个设置在此页面隐藏的功能。（弹出层 需要设计两个动态面板，一个做遮罩层（不透明度 50%，填充颜色为灰色），一个做弹出层）
  3. 中继器：可以实现动态增加，删除行的效果。
  4. 热区：可以结合动态面板用来制作日期控件的选择

一、制作弹出层页面
在制作网页过程中，为了给开发人员一个更直观的产品介绍，通常要利用 html 原型进行演示。在原型生成过程中，弹出窗口效果时非常常见的，下面记录本人制作弹出窗口效果的心得。
axure 利用动态面板制作弹出窗口效果：

  1. 在页面要弹出动态面板的位置画一个动态面板，编辑动态面板，设为隐藏模式，面板变为淡黄色。
  2. 设置点击时弹出对话框的按钮一个事件，OnClick 为 show panel（s），选择刚才的 panel 面板。
  3. 双击动态面板弹出动态面板管理页面，编辑标签，双击 state 进入新窗口，在新窗口中虚线范围内编辑要弹出的窗口的效果。
  4. 在窗口中编辑动作，保存，生成原型。
